# YHODA Project Board

## Overview

This project board tracks the Phase 2 implementation of the YHODA ETL pipeline - automating data ingestion for ~40 priority datasets across 22 Yorkshire Local Authority Districts.

## Phases

1. Discovery & Design - API credential registration, database setup, geography layer, shared pipeline plumbing
2. Prototype Build - Nomis, ONS, and DWP connectors + first 3 economy flows wired end-to-end
3. Prototype Review - Integration testing, data quality checks, stakeholder demo
4. Scale-out - Remaining 5 data sources, 10 remaining flows, orchestrator
5. Hardening & Testing - Error handling, validation edge cases, full test suite
6. Documentation - Runbooks, data dictionary, monitoring/alerting setup
7. Training & Handover - Team training sessions, final review, project close
