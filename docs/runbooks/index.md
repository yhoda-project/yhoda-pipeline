# Runbooks

A runbook is a step-by-step guide for a specific situation. Use this section when something has gone wrong or needs attention.

---

| Runbook | When to use it |
|---------|---------------|
| [A flow is failing](flow-is-failing.md) | You received an ERROR email, or a flow shows as failed in the Prefect UI |
| [Data looks wrong](data-looks-wrong.md) | A figure in the dashboard looks incorrect or a dataset appears out of date |
| [Pipeline has stopped running](pipeline-not-running.md) | No flows have run on their expected schedule |
| [API key has expired](api-key-expired.md) | A flow fails with an authentication error, or you need to rotate credentials |
